# Sokoban Worlds
A JavaScript implementation of the [Sokoban](https://en.wikipedia.org/wiki/Sokoban) game, created using [p5.js](https://p5js.org/) and the [p5.quadrille.js](https://objetos.github.io/p5.quadrille.js/) library.
